## Bitrix24_DepartmentWorktime


Отчёт для руководителя подразделения по рабочему времени соотрудников своего подразделения - Битрикс24 (коробочная версия).

1) Программа проверят, имеет ли право сотрудник сделать отчет (входит ли в заданную группу группу - константа BOSS_GROUP в include/constants.php)
2) Если да, то формируется отчёт по рабочему времени сотрудников подразделения, руководитель которого запустил данный отчёт.
3) Период по-умолчанию: с начала текущего месяца по вчерашний день. Период можно изменить на любой необходимый..
4) Выводятся все данные из таблицы `b_timeman_entries`и сравниваются с рабочим графиком сотрудника, определённым на портале).
5) Если сотрудник не отметился в данный день - запись пустая.
6) Все нарушения выделяются красным цветом.

BestTimanUser - extends CTimeManUser - Класс для работы с графиком работ.
Boss Class - класс руководителя для работы с отчётами по рабочему времени.
